Android Studio 运行错误 Execution failed for task app:preDexDebug

Error:Execution failed for task ‘:app:preDexDebug’.

com.android.ide.common.internal.LoggedErrorException: Failed to run command:

D:\Android\adt-bundle-windows\sdk\build-tools\21.1.1\dx.bat --dex --output D:\Android\studio-workspace\ManYouBang\app\build\intermediates\pre-dexed\debug\classes-2af6df9d4489af2a60a56b312b166bfa7882943d.jar D:\Android\studio-workspace\ManYouBang\app\build\intermediates\exploded-aar\com.android.support\appcompat-v7\21.0.2\classes.jar

Error Code:

1

Output:

'""' 不是内部或外部命令，也不是可运行的程序

或批处理文件。

解决方法：

找到Sdk目录下build-tools\21.1.1\dx.bat文件打开（21.1.1对应自己的构建版本号）；找到set java_exe=设置为自己的java jdk目录。

然后把类似如下代码删除

if exist “%~dp0..\tools\lib\find_java.bat” call “%~dp0..\tools\lib\find_java.bat”

if exist “%~dp0....\tools\lib\find_java.bat” call “%~dp0....\tools\lib\find_java.bat”

if not defined java_exe goto :EOF
